# retrieval_chatbot

## 部署:
* 目录cluster_model, data, index, model, vec_model为存放各类模型数据的目录, 需要从各目录中指定路径下载所需文件放在该目录中
* 安装依赖库: `pip3 install -r requirements.txt`

## 启动方法:
* 启动命令: `nohup python3 server.py &`
* **注意: 该方法服务会监听在10086端口**:

## 环境要求:
* Python3
